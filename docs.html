متـودهای کتابخانه Belrub

- ارسـال پیـام

ببینید ما در لاین اول کتابخونه رو ایمپورت کردیم و در لاین بعدیش شما باید به جای AUTH-KEY اوث اکانت خودتون رو وارد کنید.
سپس در لاین بعدی که متود ارسال پیام هستش جای target-GUID گویید جایی که قراره پیام ارسال بشه رو وارد کنید که میتونه گویید یک کاربر یا کانال یا یک گروه باشه که میتونید بطور دلخواه گویید موردنظرتون رو قرار بدید!
و در آخر پیام دلخواهتون رو به جای Hi BeLectron بزارید و کد هارو اجرا کنید تا پیامتون ارسال بشه !

from Belrub import Token

bot = Token("AUTH-KEY")
bot.sendMessage("target-GUID", "Hi BeLectron !")

sendMessage #

- ارسـال عکس

خب رفقا دقت کنید که توی این متود جای AUTH-KEY اوث اکانت رو وارد کنید و سپس جلوی متغیر address آدرس عکس موردنظرتون رو وارد کنید و جلوی متغیر caption کپشن موردنظرتون رو وارد کنید و جای target-GUID گویید جایی که میخواید عکس ارسال بشه رو وارد کنید.

from Belrub import Token

bot = Token("AUTH-KEY")

address = "example.jpg"
cap = "your-caption"
bot.sendPhoto("target-GUID", address, caption = cap)

sendPhoto #
